Feedback Analyser

D'avril à juin 2018, j'ai effectué mon stage de fin de DUT dans l'Université de Canterbury, Christchurch, Nouvelle-Zélande. Le département formant les futurs ingénieurs dispense aux étudiants des cours où ils peuvent apprendre à travailler en équipe. L'objectif de mon stage fut de développer un outil en Python permettant d'analyser des résultats de questionnaires auxquels ont répondu les étudiants de ce cours. Dans ces textes, les étudiants écrivent ce que leurs coéquipiers ont fait de bien, et ce qu'ils doivent améliorer pour leurs futurs projets. J'ai ainsi pu utiliser des outils de machine learning afin de relever automatiquement les étudiants qui ont eu des problèmes durant leurs projets

Compétences acquises :
Python : langage de programmation
Ruby on Rails : framework web utilisant le Ruby
PyCharm : IDE pour Python
LaTeX : langage de composition de documents
Sklearn : librairie de machine learning
NLTK : librairie d'analyse sémantique
Pandas : librairie d'analayse de données

Application Symfony

L'Université de La Rochelle a sollicité notre IUT afin de développer un back-office pour une application déjà existante. En effet, celle-ci ne pouvait pas être modifiée à distance et un changement dans l'application, même minime, nécessitait de remettre à jour l'application sur Android et IOS. Nous avons ainsi développer ce back-office en utilisant le Framework Symfony ainsi que du Javascript.

Compétences acquises :
Symfony : framework PHP
Javascript : langage de programmation web
MySQL : système de gestion de bases de données relationnelles
Scrum : méthode agile de gestion de projet
PHPStorm : IDE pour PHP

La bête du Gévaudan

Durant notre deuxième année de DUT, nous avons du réaliser un jeu multijoueur en Javascript. Nous avons décidé de développer un jeu en "tour par tour" où plusieurs chasseurs traquent une "bête" d'apparence inoffensive, mais qui la nuit devient très aggressive. Nous avons donc développer notre jeu en Javascript, en utilisant un serveur NodeJS. Nous avons implémenté notre application dans le framework Angular2.

Compétences acquises :
Javascript : langage de programmation web
Angular2 : framework Javascript
NodeJS : serveur Javascript
JQuery : librairie Javascript
Socket.io : librairie Javascript permettant l'échange serveur/client en temps réel

Hodena

Chaque semaine, l'IUT est chargé de planifier un emploi du temps pour chaque classe. Gérer la capacité des salles, leur équipement, etc. est très difficile et prend beaucoup de temps. On nous a donc demandé de créer une application en Java permettant la généartion automatique des emplois du temps. Nous avons utilisé du Java FX pour la partie graphique de notre application.

Compétences acquises :
Java : langage de programmation orienté objet
JavaFX : outil MVC Java de création d'interface graphique
